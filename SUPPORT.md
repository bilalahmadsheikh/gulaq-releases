# Support & bug reports

Found a bug or have a problem with **gulaq** (the desktop app or the Chrome extension)? We'd love to hear about it.

## 🐞 Report a bug

[**Open a new issue →**](https://github.com/bilalahmadsheikh/gulaq-releases/issues/new?template=bug_report.md)

When you report a bug, please include:

- **Which app** — the Chrome extension, the desktop app (gulaq Desktop Vault), or both
- **Versions** — extension version (from `chrome://extensions`), desktop app version (Settings → About), and your Windows version
- **Steps to reproduce** — what you clicked / did
- **What you expected** vs. **what actually happened** (screenshots help a lot)

The issue form will walk you through these fields.

## 💡 Feature requests

Have an idea? [Open an issue](https://github.com/bilalahmadsheikh/gulaq-releases/issues/new) and describe what you'd like.
